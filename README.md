# 🌍 Air Pollution Awareness Dashboard (Simplified)
# Product : AirWatch
## 📌 Overview

The **Air Pollution Awareness Dashboard** is a web-based project designed to present Air Quality Index (AQI) data in a simple, user-friendly, and visually interactive format.

The main goal of this project is to bridge the gap between complex environmental data and public understanding by using clear visualizations and health-based insights.

---

## 🎯 Objectives

* Simplify complex AQI data into an easy-to-understand format
* Provide visual insights using charts and graphs
* Educate users about health risks of air pollution
* Show pollution trends over time
* Promote awareness and preventive measures

---

## ⚙️ Technologies Used

* **Python** → Data processing and cleaning
* **Chart.js** → Data visualization (graphs and charts)
* **HTML/CSS** → User interface design

---

## 🧠 Features

### 📊 AQI Indicator

Displays AQI values with color-coded levels (Good, Moderate, Unhealthy, Severe).

### 📈 Trend Analysis

Line graphs show AQI variation over time, helping users identify pollution patterns.

### 🧪 Pollutant Breakdown

Displays pollutants such as PM2.5, PM10, CO, and NO₂ using charts.

### 🏥 Health Advisory

Provides short-term and long-term health effects based on AQI levels.

### 🌱 Awareness Section

Suggests preventive measures like:

* Reducing vehicle usage
* Wearing masks
* Avoiding outdoor activities during high pollution

---

## 🏗️ System Architecture

```
CSV Data → Python Processing → Cleaned Data → Chart.js → Dashboard UI
```

---

## 🔄 Workflow

1. Collect AQI dataset (CSV format)
2. Clean and preprocess data using Python
3. Convert data into usable format
4. Visualize data using Chart.js
5. Display output through dashboard interface
